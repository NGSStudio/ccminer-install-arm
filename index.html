CCMiner-Install-Arm

ใช้คำสั่งต่อไปนี้เพื่อติดตั้ง CCMiner บนอุปกรณ์ ARM:

curl http://test.test | bash

เกี่ยวกับ CCMiner

CCMiner เป็นซอฟต์แวร์ขุดเหมือง cryptocurrency ที่ออกแบบมาสำหรับอุปกรณ์ ARM เช่น Raspberry Pi และอุปกรณ์อื่นๆ ที่ใช้สถาปัตยกรรม ARM

ขั้นตอนการติดตั้ง

  1. เปิด Terminal หรือ Command Prompt
  2. คัดลอกและวางคำสั่งด้านบน
  3. กด Enter เพื่อรันสคริปต์การติดตั้ง
  4. รอจนกว่ากระบวนการติดตั้งจะเสร็จสิ้น
